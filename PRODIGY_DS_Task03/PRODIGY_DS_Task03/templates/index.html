ℹ️ About This Project
📌 Objective

Predict whether a bank customer will subscribe to a term deposit product based on demographic and behavioral data from marketing call campaigns.

📦 Dataset

UCI Bank Marketing Dataset — features include age, job, marital status, education, account balance, contact details, and campaign history.

🛠️ Tech Stack

Python · scikit-learn · pandas · matplotlib · seaborn · Flask · HTML/CSS/JS

🌳 Model

Decision Tree Classifier with Gini criterion, max depth 5, minimum sample constraints to prevent overfitting.
